# Foundation Plan

## Objectif

Mettre en place progressivement le socle technique du monorepo sans perdre les decisions metier structurantes.

## Etapes ciblees

1. Finaliser documentation, ADR et regles metier.
2. Preparer la structure repository.
3. Ajouter l'orchestration Docker Compose lors d'une tache dediee.
4. Initialiser le backend Django 5.2 LTS lors d'une tache dediee.
5. Initialiser le frontend React TypeScript strict avec Vite lors d'une tache dediee.
6. Ajouter qualite, tests et CI lors de taches dediees.
7. Ajouter les premiers health checks.
8. Valider le gate Foundation.

## Contraintes structurantes

- Titan exclut definitivement local et service.
- Les materiels sont partages entre Hahitantsoa et Titan.
- La confirmation de reservation devra etre transactionnelle.
- Les documents sensibles devront etre prives et controles.
