# Architecture

Ce dossier documente l'architecture cible du projet Hahitantsoa / Titan.

La stack cible comprend Django 5.2 LTS, Django REST Framework, React, TypeScript strict, Vite, PostgreSQL, Celery, Redis, Celery Beat, Django Channels, Flower non public, Nginx, ASGI et OpenAPI avec drf-spectacular.

La mission actuelle est documentaire. Aucun composant technique n'est encore implemente.
